Update ubuntu

25 Jun 2016

You can manually update via terminal by running:

$ sudo apt-get update
$ sudo apt-get upgrade

Additionally you can run:

$ sudo apt-get dist-upgrade

This will clean up any packages that are no longer needed:

$ sudo apt-get clean
$ sudo apt-get autoremove

How to change mouse speed/sensitivity?

Jun 2016

"Razer Razer DeathAdder" - mouse name.

$ xinput --set-prop "Razer Razer DeathAdder" "Device Accel Constant Deceleration" 4

Чтобы посмотреть подключенные устройства:

$ xinput

Install Latest Arc GTK Theme on Ubuntu 16.04

Jun 2016

Getting the latest version of the Arc GTK theme on Ubuntu 16.04 is simple. You have a number of choices available.

You can add the theme designer’s personal repository to get new versions of Arc on Ubuntu 16.04 LTS as and when they’re released. To add the Arc GTK repo run:

$ sudo sh -c "echo 'deb http://download.opensuse.org/repositories/home:/Horst3180/xUbuntu_16.04/ /' >> /etc/apt/sources.list.d/arc-theme.list"

Then, refresh your software sources and install the theme:

$ sudo apt-get update

$ sudo apt-get install arc-theme

You’ll also need to import the repo’s key to quite any command-line errors:

$ wget http://download.opensuse.org/repositories/home:Horst3180/xUbuntu_16.04/Release.key

$ sudo apt-key add - < Release.key
